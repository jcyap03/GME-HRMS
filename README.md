# GME HRMS Professional v4.1 PWA

## Important privacy warning
This repository must contain only the application files. Do **not** upload employee backup JSON files, receipts, contracts, IC details, salary data, or other employee records to a public GitHub repository.

## Publish using GitHub Pages
1. Create a new GitHub repository, for example `GME-HRMS`.
2. Upload everything inside this folder, keeping the `icons` folder structure.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then save.
6. Open the Pages URL shown by GitHub.

## Restore your existing information
The application data is stored separately in each browser. On your phone, open **Backup Manager → Restore JSON** and select your latest private backup file.

## Install on iPhone / iPad
Open the GitHub Pages URL in Safari → Share → Add to Home Screen.

## Install on Android / desktop Chrome
Open the site and use the **Install App** button or browser installation icon.
